# Serbizyu — Industry Coverage Matrix
*Every industry that plausibly shows up in a Philippine local economy, mapped to a fulfillment archetype, with demand signals, marketing angles, and honest flags for what this platform should NOT chase.*

---

## How to Read This

- **Archetype** references the Fulfillment Archetype Library (A1–A10).
- **Readiness** is a build verdict: `preset` (works today), `config` (tier-2 template tweak), `build` (needs Phase 3 work), `avoid` (wrong platform).
- **Marketing angle** is the one-line hook for that category's distribution content.
- **Hotspot** flags expected demand density in a provincial town like Candon.

## 10. Explicit Exclusions (What NOT to Chase)

| Category | Why excluded |
|---|---|
| Ride-hailing at scale (Grab-style) | Requires real-time dispatch optimization, driver supply liquidity in the hundreds, surge pricing. A2 covers local dispatch; that is enough. Do not rebuild Grab. |
| Food delivery fleet (Foodpanda-style) | Restaurant marketplace + driver fleet + sub-hour SLAs. Different business. A2 covers *single-vendor* delivery fine. |
| E-wallet / money services | BSP-regulated. Xendit handles money movement; we never touch stored value. |
| Professional licensed practice (medicine, law, engineering sign-off) | Liability and regulation. Platform can host *logistics* around these (booking a vet visit) but not the licensed act itself. |
| Real estate sales | Transaction values too high for Xendit escrow comfort; legal conveyancing required. Rentals of *equipment* yes, property no. |
| Long-haul logistics / freight forwarding | Multi-leg, customs, warehousing. Out of scope for a hyper-local platform. |
| Job placement / employment agency | DOLE licensing for recruitment. Gigs and bookings yes, employment contracts no. |

## Coverage Summary

| Sector | Dominant archetype(s) | Launch readiness | Strategic value |
|---|---|---|---|
| Home & Property | A8, A3, A1 | **Preset** | Anchor sector |
| Transport | A2 | Config (flagship case) | Proof of engine flexibility |
| Food | A4 | **Preset** | Volume driver |
| Personal Care | A3 | **Preset** | Repeat-rate driver |
| Events | A5, A1 | Config | Seasonal spikes |
| Professional/Digital | A9, A10 | **Preset** | Margin driver |
| Agriculture | A7 | Config | Rural differentiation |
| Retail/Goods | A4 | Preset (+Phase 3 for requests) | Completes Offer model |
| Gov/Community | A1, A7 | Config | Legitimacy + LGU partnerships |

**The punchline:** four preset archetypes (A1, A3, A4, A9) plus one configured flagship (A2 tricycle) cover roughly 80% of a provincial service economy by transaction count. The remaining 20% is tier-2 configuration and Phase 3 tool builds, not new engineering.
